# Ecommerce-SyntaxWear

## Descrição

Projeto estático de uma landing page para loja de tênis "SyntaxWear".
A página usa HTML e CSS puro para estruturar e estilizar:
- cabeçalho com navegação principal e ícones de acesso rápido
- seção hero com chamada para ação
- grades de categorias de produtos
- seção de produtos em destaque
- rodapé com newsletter e links secundários

## Como abrir

1. Abra o arquivo `index.html` no navegador.
2. Ou use uma extensão como Live Server no VS Code.

## Estrutura de pastas

- `index.html` - página principal.
- `assets/css/` - estilos individuais por seção:
  - `reset.css` - reset básico de CSS.
  - `variables.css` - variáveis globais de cores e tipografia.
  - `base.css` - regras globais e utilitários.
  - `header.css` - estilo do cabeçalho e navegação.
  - `hero.css` - estilo da seção hero.
  - `product-category.css` - estilo das categorias de produtos.
  - `featured-products.css` - estilo dos cards de produtos em destaque.
  - `footer.css` - estilo do rodapé.
- `assets/images/` - imagens usadas no projeto:
  - `banners/`, `favicons/`, `icons/`, `logo/`, `products/`
- `docs/conventions.md` - convenções de nomeação e organização do projeto.

## Tecnologias

- HTML5
- CSS3
- Sem JavaScript nesta versão

## Notas importantes

- O arquivo `assets/css/styles.css` está presente mas sem conteúdo.
- O projeto foi estruturado para ser modular e fácil de manter.
- Para melhor responsividade, verifique as media queries existentes.
